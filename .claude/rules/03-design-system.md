# Design System Rules

## General Direction
- Build a smooth, premium, modern B2B website
- The website should feel calm, capable, and technically refined
- Use a dark blue visual direction with clean surfaces and strong contrast
- Use Geist as the main website font
- Prioritize a polished and intentional layout over flashy effects

## Colors
- Never use default Tailwind blue or indigo as the primary brand color
- Derive all main colors from the HG Automations brand system
- Use subtle borders and controlled contrast
- Avoid harsh pure black on pure white combinations

## Typography
- Headings should feel compact, confident, and high-clarity
- Body text should be highly readable with generous line-height
- Keep typographic hierarchy strong and obvious
- Avoid oversized marketing text that feels cheap or exaggerated

## Spacing
- Use consistent spacing tokens across the entire site
- Keep sections breathable and structured
- Avoid cramped card layouts or random spacing jumps

## Surfaces and Depth
- Use layered depth: base surface, elevated cards, floating emphasis elements
- Prefer subtle tinted shadows and soft borders over generic flat shadows

## Motion
- Motion should feel smooth and restrained
- Animate only transform and opacity
- Never use `transition-all`
- Avoid decorative animation that does not improve clarity

## Interactive States
- Every clickable element must have hover, focus-visible, and active states
- Accessibility must be built in from the start

## Visual Tone
- Do not make the interface playful, startup-gimmicky, or overly futuristic
- Do not overload the hero with visual noise
- Keep the design elegant, sharp, and conversion-oriented
