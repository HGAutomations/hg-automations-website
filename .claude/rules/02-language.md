# Language Rules

- All final user-facing content must be written in German
- This includes navigation, headlines, subheadings, body text, buttons, forms, placeholders, validation messages, and footer content
- English is allowed only in code, comments, filenames, and technical documentation
- Do not mix English and German in the visible UI
- Copy must be clear, concise, professional, and B2B-oriented
- Avoid hype-heavy AI buzzwords if they reduce clarity
